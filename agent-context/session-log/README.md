# Session Logs

Use one branch-scoped log file per active branch.

Direct work on `main` may use `main.md`. Feature branch logs should use `YYYY-MM-DD-feature-branch.md`, using the best-known branch creation date.

Each log should include:

- `agent`
- `branch`
- `head`
- `summary`
- `validation`
- `follow-ups`

Update the current branch log before committing.
